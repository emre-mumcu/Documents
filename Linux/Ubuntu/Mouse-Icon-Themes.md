
# Linux Mint Mouse Icon Themes

https://github.com/linuxmint/mint-cursor-themes
https://github.com/KDE/breeze


For system-wide installation:

sudo cp -r cursors-folders /usr/share/icons/ 

For a single user installation:

mkdir -p ~/.icons
cp -r mint-x-cursors ~/.icons/

Breeze:

sudo apt install breeze-cursor-theme
